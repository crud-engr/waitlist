# waitlist

#### Clone the project
```git clone url```

#### Navigate to the folder
```cd folder_name```

#### Install Dependencies
```npm install```

#### Create .env file and paste the following
```NODE_ENV=development```
```PORT=3300```
```LOCAL_DB='mongodb://localhost:27017/waitlist'```

#### Run the app
```npm start```
